# <div align="center">ChatWitt-System</div>
<div align="center">The system Discord bot to the server 'ChatWitt'</div>
<div align="center">Basic commands and event listeners.</div>
<br><br>

### Download the [interactions](https://github.com/interactions-py/interactions.py) library, for using the basics.
```
pip install -U discord-py-interactions
```
<br>

### Install [interactions tasks](https://github.com/Catalyst4222/interactions-tasks) for task integration.
```
pip install interactions-tasks
```
<br>

### This project uses 'yaml' files, so it's needed to install pyyaml.
```
pip3 install pyyaml
```
