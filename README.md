# ChatWitt-System
The system Discord bot to the server 'ChatWitt'
Basic commands and event listeners.

To use the [interactions](https://github.com/interactions-py/interactions.py) library, you first need to install it.
```
pip install -U discord-py-interactions
```


To use the [interactions tasks](https://github.com/Catalyst4222/interactions-tasks), you also need need to install it first.
```
pip install interactions-tasks
```

This project also uses 'yaml' files, so need to install pyyaml.
```
pip3 install pyyaml
```
