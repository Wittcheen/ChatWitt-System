# ChatWitt-System
The system Discord bot to the server 'ChatWitt'
<br/><br/>
A Discord bot that is programmed to perform a wide range of functions

- Join and leave listener
- Update member count channel
- React roles system with buttons
- Advanced giveaway system
- Database to store giveaways
- Poll system with select menu
- Rules, news and clear commands
- Twitch auto live notifications
- Tickets / Twitch mod application

<br/>

### Install all the necessary librarys
[interactions](https://github.com/interactions-py/interactions.py) and [interactions-tasks](https://github.com/Catalyst4222/interactions-tasks)
```
pip install -U discord-py-interactions
pip install interactions-tasks
```
```
pip3 install pyyaml
pip install python-dotenv
pip install requests
pip install aiomysql
pip install asyncio
```
